## Repository Structure
```text
LeetCode-Solutions/
├── Easy/
├── Medium/
├── Hard/
└── README.md
```
Each folder contains problems organized by **LeetCode difficulty**.
Every problem has its own directory named using the format:
`<problem-number>-<problem-name>/`
Example:
```text
Easy/
└── 0268-Missing-Number/
    └── solution.cpp
```
